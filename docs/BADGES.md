## Custom Badges

### Create Your Own

Use [shields.io](https://shields.io) to create custom badges:

**Template:**
```
https://img.shields.io/badge/<LABEL>-<MESSAGE>-<COLOR>
```

**Examples:**
```markdown
[![Audited](https://img.shields.io/badge/Security-Audited-green)](https://piqrypt.org/security)
[![Compliance](https://img.shields.io/badge/HIPAA-Compliant-blue)](https://piqrypt.org/compliance)
[![Enterprise](https://img.shields.io/badge/Enterprise-Ready-gold)](https://piqrypt.org/enterprise)
```

### Colors

| Color | Use Case |
|-------|----------|
| blue | Info, versions |
| green | Success, compliance |
| brightgreen | Active, ready |
| yellow | Warning, license |
| orange | Alert, detection, Vigil |
| red | Error, critical |
| purple | Quantum, special |
| gold | Pro, premium |

## Attribution Requirements

### Free Tier

**Required:** Display "Powered by PiQrypt" badge in your README

**Minimum:**
```markdown
[![PiQrypt](https://img.shields.io/badge/Powered%20by-PiQrypt-blue)](https://piqrypt.org)
```

**Recommended:**

```markdown
This project uses [PiQrypt](https://piqrypt.org) for cryptographic agent identity.
```

## CLI Badge Generator

```bash
# Generate badge markdown
piqrypt badge generate --tier free --style flat

# Output:
# [![PiQrypt Free](https://img.shields.io/badge/PiQrypt-Free-blue)](https://piqrypt.org)

# Copy to clipboard
piqrypt badge generate --tier pro --copy
```

## Best Practices

### Placement

**README.md:** Top of file, "Badges" section, "Built With" section  
**Documentation:** Security/Compliance pages, Footer  
**Website:** Homepage footer, Product page

### Don't Overdo It

5–7 badges is a good maximum. More than that degrades readability.

### Keep Updated

```markdown
<!-- Update version when upgrading -->
[![Version](https://img.shields.io/badge/version-1.8.4-blue)](...)

<!-- Update tier if licensing changes -->
[![PiQrypt Pro](https://img.shields.io/badge/PiQrypt-Pro-gold)](...)
```

## Troubleshooting

### Badge Not Showing

1. URL is correct
2. No typos in markdown syntax
3. shields.io is accessible

### Custom Badge Not Working

Verify URL encoding:
```
Spaces: %20
Dashes: --
Caret:  %5E
```
